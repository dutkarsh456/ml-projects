# ML Projects

Two beginner ML projects built with Python and scikit-learn.

---

## 1. Customer Service Chatbot
- Rule-based NLP chatbot
- Detects 8 intents: billing, orders, tech support, refunds, etc.
- Built with HTML + JavaScript
- File: `chatbot.html`

## 2. Spam Email Classifier
- Classifies emails as SPAM or HAM
- Uses Naive Bayes + TF-IDF vectorization
- Accuracy: ~80%
- File: `spam_classifier.py`
